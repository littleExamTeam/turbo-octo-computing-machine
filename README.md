git指令
修改之后需要先进行如下操作将修改提交到本地
git add .
git commit -m "这里写清楚你改的东西 然后写上自己的名字简称（主要是为了区分是谁提交的）"

提交到本地之后，就可以提交到远程仓库，输入如下指令
git pull (这里是为了合并远程仓库中，合并之后可能会会叫你修改代码，因为可能有冲突，按照提示修改代码，然后进行提交到本地操作，就是上面那两条指令)
git push (这个就是提交指令)
